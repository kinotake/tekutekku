# tekutekku
☆環境構築</br>

1.git clone git@github.com:kinotake/tekutekku.git</br>
2.env.exampleの名前を.envに変更</br>
3.cd ディレクトリまでのパス</br>
4.docker compose up -d --build</br>
5.docker compose exec php bash</br>
6.composer install</br>
実行環境　http://localhost:81

☆使用技術（実行環境）</br>
laravel 8.83.27</br>
php 7.4.9</br>
